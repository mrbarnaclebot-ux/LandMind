  <files>packages/server/src/routes/earnings.ts</files>
  <action>
Create earnings.ts router with three endpoints:

**GET /api/earnings** (authenticated)
- Requires JWT auth (use requireAuth middleware)
- Get userId from req.user.id
- Call getEarningsForUser(userId)
- Return JSON: { weightedScore, sharePercent, totalClaimed, claimable, projectedDaily, rank, percentile }
- Include rank and percentile from leaderboardService

**POST /api/earnings/claim** (authenticated)
- Requires JWT auth
- Check claimable amount >= MIN_CLAIM (0.025 SOL)
- Generate Merkle proof for user's share
- Build unsigned transaction calling claim_earnings instruction
- Serialize transaction to base64
- Return: { transaction, amount, proof }
- Client will sign and send

**POST /api/earnings/confirm** (authenticated)
- Requires JWT auth
- Body: { signature, amount }
- Verify transaction signature on-chain (optional, or trust client)
- Record Claim in database
- Update user's EarningsSnapshot.totalClaimed
- Emit claim:success socket event to user's room
- Return: { success: true, newBalance }

**Implementation pattern:**
```typescript
import { Router } from 'express';
import { requireAuth } from '../middleware/auth.js';
import { getEarningsForUser, updateUserEarningsSnapshot } from '../services/earningsService.js';
import { getUserRank, getUserPercentile } from '../services/leaderboardService.js';
import { generateProof, generateMerkleTree } from '../services/merkleService.js';
import { getIO } from '../lib/socket.js';

export const earningsRouter = Router();

earningsRouter.get('/', requireAuth, async (req, res) => {
  const userId = req.user!.id;
  const earnings = await getEarningsForUser(userId);
  const user = await prisma.user.findUnique({ where: { id: userId } });
  const rankInfo = await getUserRank(user!.walletPubkey);
  const percentile = await getUserPercentile(user!.walletPubkey);

  res.json({
    ...earnings,
    weightedScore: earnings.weightedScore.toString(),
    totalClaimed: earnings.totalClaimed.toString(),
    claimable: earnings.claimable.toString(),
    projectedDaily: earnings.projectedDaily.toString(),
    rank: rankInfo.rank,
    percentile,
  });
});
```

Use BigInt.toString() for JSON serialization (BigInt not JSON-safe).

Create leaderboard.ts router:

**GET /api/leaderboard** (public, optional auth)
- Call getTopUsers(10) for top 10
- If authenticated, also get user's rank and score
- Return: { top10: [...], userRank?, userScore?, totalUsers }

**Implementation:**
```typescript
import { Router } from 'express';
import { optionalAuth } from '../middleware/auth.js';
import { getTopUsers, getUserRank, getTotalUsers } from '../services/leaderboardService.js';

export const leaderboardRouter = Router();

leaderboardRouter.get('/', optionalAuth, async (req, res) => {
  const top10 = await getTopUsers(10);
  const totalUsers = await getTotalUsers();

  const response: any = {
    top10: top10.map(entry => ({
      ...entry,
      score: entry.score.toString(),
    })),
    totalUsers,
  };

  // Include user's rank if authenticated
  if (req.user) {
    const user = await prisma.user.findUnique({ where: { id: req.user.id } });
    if (user) {
      const rankInfo = await getUserRank(user.walletPubkey);
      response.userRank = rankInfo.rank;
      response.userScore = rankInfo.score.toString();
    }
  }

  res.json(response);
});
```

**Add optionalAuth middleware** (if not exists):

- Sets req.user if valid token present, otherwise req.user = null

Add the following models to the Prisma schema:

1. **FeeDeposit model** - tracks incoming fee deposits:

   - @@map("fee_deposits")

2. **Claim model** - tracks user claims:
   - id: String @id @default(uuid())
   - user: User @relation(fields: [userId], references: [id])
   - userId: String @map("user_id")
   - amount: BigInt (lamports claimed)
   - txSignature: String @unique @map("tx_signature")
   - claimedAt: DateTime @default(now()) @map("claimed_at")
   - @@map("claims")

3. **FeeSource enum**: DEPLOYMENT, PUMPFUN

Add `claims Claim[]` relation to User model.

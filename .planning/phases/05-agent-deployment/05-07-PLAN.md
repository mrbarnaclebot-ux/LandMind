**Pre-requisites:**
1. Ensure .env is configured with:
   - HELIUS_RPC_URL or devnet RPC
   - SERVER_WALLET_SECRET (base58 keypair with SOL for minting fees)
   - MERKLE_TREE_ADDRESS (run createMerkleTree.ts first if not set)

2. Start services:
   ```bash
   docker compose up -d  # PostgreSQL + Redis
   cd packages/server && npm run dev
   cd packages/client && npm run dev
   ```

**Test Flow:**

1. **Wallet Connection**
   - Open http://localhost:5173
   - Connect wallet (Phantom/Solflare on devnet)
   - Verify "DEPLOY AGENT" button appears in header

2. **Deploy Agent (CONTRACT-01, AGENT-01)**
   - Hover over DEPLOY AGENT - should show "COST: 0.1 SOL" tooltip
   - Click DEPLOY AGENT
   - Wallet popup should appear requesting 0.1 SOL
   - Approve transaction
   - Toast should show: "Preparing..." -> "Signing..." -> "Sending..." -> "Confirming..." -> "Agent deployed!"

3. **Agent Appears on Grid (AGENT-02, WORLD-03)**
   - After successful deploy, camera should pan (or agent should be visible)
   - Agent should appear as small green box on a hex
   - Agent should have subtle bobbing animation

4. **Agent Dashboard (AGENT-05)**
   - Click "MY AGENTS" button in header
   - Dashboard should slide in from left
   - Should show:
     - Summary: "AGENTS: 1/1", "MINING: ACTIVE"
     - Total mined resources (starts at 0)
     - Agent card with: "AGENT #1", "MINING" status, hex coordinates
   - Click "LOCATE" button - panel should close (camera pan logged)

5. **Agent Mining (verify simulation)**
   - Wait ~30 seconds
   - Open dashboard again
   - Resource totals should have increased (Gold/Silver/Copper/Iron)
   - Agent card should show updated resources

6. **Relocation (WORLD-04)**
   - This may take longer depending on hex resource depletion
   - If hex depletes, agent should:
     - Change status to "RELOCATING"
     - Move to new hex with animation
     - Resume "MINING" status
   - (May need to use dev endpoint to trigger depletion for testing)

7. **Multiple Agents**
   - Deploy a second agent
   - Verify both appear on grid
   - Verify dashboard shows both agents
   - Verify summary shows "AGENTS: 2/2"

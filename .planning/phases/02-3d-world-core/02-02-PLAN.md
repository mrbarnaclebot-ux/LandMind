<task type="auto">
  <name>Task 3: Create terrain generator</name>
  <files>packages/client/src/terrain/terrainGenerator.ts</files>
  <action>
Create terrainGenerator.ts with noise-based procedural generation:

1. Import:
   - createNoise2D from 'simplex-noise'
   - alea from 'alea'
   - getBiome, Biome from './biomes'
   - hexToPixel from '../hex/hexMath' (will exist from 02-01)

2. Define HexData interface:
   ```typescript
   interface HexData {
     q: number;
     r: number;
     elevation: number; // 0, 1, or 2 (tiers)
     biome: Biome;
   }
   ```

3. Define TerrainSeed type: { elevation: string; moisture: string }

4. Implement generateHexData(gridRadius: number, seed?: TerrainSeed):
   - Default seed: { elevation: 'elevation-default', moisture: 'moisture-default' }
   - Create two noise functions with different seeds
   - Generate hexes in a spiral pattern from center:
     - For gridRadius=30, iterate q from -30 to 30, r from -30 to 30
     - Filter to keep only hexes where hexDistance from (0,0) <= gridRadius
   - For each hex:
     - Get world position via hexToPixel(q, r)
     - Generate elevation noise with fractional Brownian motion (3 octaves)
     - Quantize to 0/1/2 based on thresholds (< 0.4 = 0, < 0.7 = 1, else 2)
     - Generate moisture noise (single octave, different scale)
     - Normalize moisture to 0-1
     - Get biome via getBiome(elevation, moisture)
   - Return HexData[]

5. Use noise scales:
   - Elevation: scale = 0.02 (large features)
   - Moisture: scale = 0.015 (slightly larger regions)
  </action>

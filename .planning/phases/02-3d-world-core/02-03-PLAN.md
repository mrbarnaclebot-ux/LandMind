1. Imports:
   - React: useEffect, useRef, useMemo
   - react-babylonjs: useScene

   - ../hex/hexMesh: createBeveledHexMesh
   - ../hex/hexMath: hexToPixel, ELEVATION_STEP
   - ../terrain/terrainGenerator: generateHexData
   - ../terrain/biomes: getBiomeColor

2. Props interface:
   ```typescript
   interface HexWorldProps {
     gridRadius?: number; // default 30 (~2700 hexes)
     seed?: { elevation: string; moisture: string };
   }
   ```

3. Component implementation:
   - useScene() to get Babylon scene
   - useMemo to generate hex data (only regenerate on seed/radius change)
   - useRef for mesh reference
   - useEffect for thin instance setup:

4. Thin instance setup in useEffect:
   ```typescript
   // Create template mesh (not rendered directly)
   const templateMesh = createBeveledHexMesh(scene, { size: 1.0, height: 0.3, bevelSize: 0.08 });
   templateMesh.isVisible = false;

   // Create instance mesh
   const instanceMesh = templateMesh.clone("hexInstances");
   instanceMesh.isVisible = true;

   // Allocate buffers
   const matrixBuffer = new Float32Array(16 * hexes.length);
   const colorBuffer = new Float32Array(4 * hexes.length);

   // Fill buffers
   hexes.forEach((hex, i) => {
     const { x, z } = hexToPixel(hex.q, hex.r);
     const y = hex.elevation * ELEVATION_STEP;
     Matrix.Translation(x, y, z).copyToArray(matrixBuffer, i * 16);

     const color = getBiomeColor(hex.biome);
     colorBuffer.set([color.r, color.g, color.b, 1], i * 4);
   });

   // Apply buffers
   instanceMesh.thinInstanceSetBuffer("matrix", matrixBuffer, 16, true);
   instanceMesh.thinInstanceRegisterAttribute("color", 4);
   instanceMesh.thinInstanceSetBuffer("color", colorBuffer, 4, true);
   instanceMesh.thinInstanceRefreshBoundingInfo(true);
   ```

5. Material setup:
   - Create StandardMaterial with vertex colors enabled
   - Set material.useVertexColors = true (NOTE: verify this works with thin instances)
   - If vertex colors don't work, fall back to a single base color for now

6. Cleanup in useEffect return:
   - Dispose template and instance meshes

7. Return null (mesh is created imperatively)

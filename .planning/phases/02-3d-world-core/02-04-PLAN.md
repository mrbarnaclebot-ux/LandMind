  <files>packages/client/src/camera/isometricCamera.ts</files>
  <action>
Create isometricCamera.ts with orthographic ArcRotateCamera:

1. Imports:
   - ArcRotateCamera, Camera, Scene, Vector3, PointerEventTypes from '@babylonjs/core'

2. Define camera configuration constants:
   ```typescript
   const DEFAULT_ALPHA = Math.PI / 4;  // 45 degrees horizontal rotation
   const DEFAULT_BETA = Math.PI / 3;   // ~60 degrees vertical (isometric feel)
   const DEFAULT_ORTHO_SIZE = 20;      // Initial zoom level
   const MIN_ORTHO_SIZE = 5;           // Maximum zoom in
   const MAX_ORTHO_SIZE = 60;          // Maximum zoom out
   const ZOOM_SPEED = 2;               // Ortho units per wheel tick
   ```

3. Implement setupIsometricCamera(scene: Scene, canvas: HTMLCanvasElement):
   - Create ArcRotateCamera with name "isoCam"
   - Set alpha, beta to defaults, radius to 100 (doesn't affect ortho)
   - Set target to Vector3.Zero()
   - Set camera.mode = Camera.ORTHOGRAPHIC_CAMERA
   - Calculate aspect ratio from canvas
   - Set ortho bounds based on DEFAULT_ORTHO_SIZE and aspect ratio:
     ```typescript
     const aspectRatio = canvas.width / canvas.height;
     camera.orthoLeft = -DEFAULT_ORTHO_SIZE * aspectRatio;
     camera.orthoRight = DEFAULT_ORTHO_SIZE * aspectRatio;
     camera.orthoTop = DEFAULT_ORTHO_SIZE;
     camera.orthoBottom = -DEFAULT_ORTHO_SIZE;
     ```
   - Configure controls:
     - camera.panningSensibility = 50 (lower = faster panning)
     - camera.wheelPrecision = 0 (disable default zoom - we handle it)
     - camera.allowUpsideDown = false
     - camera.lowerBetaLimit = 0.3 (prevent looking straight down)
     - camera.upperBetaLimit = Math.PI / 2.2 (prevent looking horizontal)
   - Attach controls: camera.attachControl(canvas, true)
   - Return camera

4. Implement handleZoom function for custom orthographic zoom:
   ```typescript
   function handleZoom(
     camera: ArcRotateCamera,
     delta: number,
     canvas: HTMLCanvasElement
   ): void {
     const currentSize = camera.orthoTop!;
     const newSize = Math.max(
       MIN_ORTHO_SIZE,
       Math.min(MAX_ORTHO_SIZE, currentSize + delta * ZOOM_SPEED)
     );
     const aspectRatio = canvas.width / canvas.height;
     camera.orthoLeft = -newSize * aspectRatio;
     camera.orthoRight = newSize * aspectRatio;
     camera.orthoTop = newSize;
     camera.orthoBottom = -newSize;
   }
   ```

5. Implement attachZoomHandler(scene: Scene, camera: ArcRotateCamera, canvas: HTMLCanvasElement):
   - Subscribe to scene.onPointerObservable
   - Filter for PointerEventTypes.POINTERWHEEL events
   - Get wheel delta from event (normalize for cross-browser)
   - Call handleZoom with normalized delta
   - Return a cleanup function that removes the observer

6. Export: setupIsometricCamera, handleZoom, attachZoomHandler
